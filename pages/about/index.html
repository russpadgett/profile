About Me

With over 3 decades of experience in software development, I've contributed to projects ranging from innovative startups to enterprise applications. My passion lies in creating software that not only works but makes a meaningful impact.

My Journey

My first lines of code were in BASIC on my Commodore 64 when I was 15 years old. A decade later I was building websites with HTML, CSS, and JavaScript which lead to my professional career as a software developer creating scalable, efficient software solutions with C# and ASP.NET. The next decade came the introduction of cloud providers and building cloud based applications with containerization followed by serverless and building serverless applications. This decade has introduced AI and building applications with AI integrations.

From my first "Hello World" to architecting cloud applications with AI integration, every step has been a learning opportunity. Continuous learning isn't just a buzzword for me; it's essential. The tech industry moves fast, and staying curious about new tools, methodologies, and best practices is what I do every day. I believe the best software is built by teams that value clean code, thoughtful design, and user-centric thinking.

Beyond Code
