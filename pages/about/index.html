About Me

Over 3 decades of software development experience with startups to Fortune 100 companies, specializing in building applications using modern cloud-based architectures, microservices, containerization with Docker/Kubernetes, and AI-powered solutions.

My Journey

I wrote my first lines of code when I was 15 years old using BASIC on my Commodore 64.

Over the years, I grew my tech stack experience with personal projects and part-time jobs, learning C, Visual Basic, and building my first websites with HTML, CSS, and JavaScript.

In 1999, I acheived the level of Senior Software Engineer working for JPMorgan Chase creating software solutions with C# and .NET.

In 2010, at Callaway Golf, I started working with Azure as a cloud provider, a few years later we were building containerized applications using Docker/Kubernetes, soon followed by serverless applications.

In 2025, AI is revolutionizing software development, and at GEICO, we are leading the way with AI-powered solutions.

From my first "Hello World" to architecting cloud applications with AI integration, every step has been a learning opportunity. The tech industry moves fast, and staying curious about new tools, methodologies, and best practices is what I do every day. I believe the best software is built by teams that value clean code, thoughtful design, and user-centric thinking.
